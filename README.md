# FDU-CS-Handwritten-Notes

复旦大学计算机科学与技术相关课程的个人手写复习笔记分享仓库。

本仓库主要收录我在课程学习与期末复习过程中整理的手写知识点总结。笔记通常会在扫描后以 PDF 形式上传，内容来源包括授课 PPT、课堂记录、个人手写整理和复习时的理解加工。

这些资料不是课程官方材料，也不替代教材、作业、实验或教师发布的复习要求。它更适合被当作一个额外参考，用来快速回顾知识框架、检查遗漏点，或从另一种整理方式中获得启发。

## 重要声明

在使用本仓库资料前，请务必知悉以下几点：

1. **仅供参考**：本仓库内容来自个人手写笔记和复习整理，内容筛选、逻辑重构、重点划分都带有主观性。
2. **以官方资料为准**：不同年份、不同授课教师的课程安排、考核重点和知识结构可能存在差异。复习时请始终以你所在学期教师发布的最新 PPT、教材、作业、实验要求和考试说明为准。
3. **可能存在错误**：受限于个人理解和整理精力，笔记中可能存在概念偏差、事实错误、条件遗漏、符号不统一、字迹不清或错别字。请不要将其作为唯一复习信源。
4. **尊重课程版权与学术诚信**：本仓库只分享个人整理后的笔记，不应上传未获授权公开传播的课程原始 PPT、试卷、答案、作业解答或其他受限材料。请勿将本仓库内容用于违反课程规定或学术诚信要求的场景。

## 当前状态

仓库正在建设中。后续会按课程逐步上传扫描版 PDF 笔记，并在课程目录中维护简要索引。

## 目录结构

```text
.
├── README.md
├── courses/
│   └── <course-name>/
│       ├── README.md
│       └── pdfs/
│           └── <topic>.pdf
```

说明：

- `courses/<course-name>/README.md`：记录课程简介、适用范围、PDF 笔记目录和维护状态。
- `courses/<course-name>/pdfs/`：存放该课程的扫描版笔记 PDF。

## 反馈与纠错

这个仓库主要是个人学习笔记分享。

如果你在阅读过程中发现概念错误、逻辑漏洞、表述不清、页码问题、文件损坏或其他需要修正的地方，欢迎通过 Issue 指出。若希望补充说明，也可以提供相关依据或建议，我会在有时间时统一核对和调整。

反馈时建议说明：

- 问题所在课程、文件和页码
- 你认为有问题的原因
- 可参考的官方资料、教材页码或课程材料位置
- 建议修改方式
